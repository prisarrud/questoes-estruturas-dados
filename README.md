# Estruturas de Dados - Lista de Exercícios

- Autores do Projeto: Letícia Pillar Lisboa e Priscila Ricardo Arruda
- Disciplina: Estruturas de Dados em Python
- Professor: Felipe de Morais

Este repositório contém 5 questões práticas relacionadas a estruturas de dados e algoritmos.

## Estrutura

Cada questão está organizada em um notebook Jupyter (.ipynb) separado:

- Questão 1 - Expressão Matemática
- Questão 2 - Atendimentos Hospitalares
- Questão 3 - Mapa do Brasil
- Questão 4 - Aventureiro
- Questão 5 - Sugestão de Arquivos

## Objetivo

O objetivo é praticar diferentes estruturas de dados e algoritmos, como:

- Pilha
- Fila de prioridade
- Grafos
- Árvores
- Busca em largura (BFS) e profundidade (DFS)

## Como usar

1. Abra os notebooks no VS Code, Jupyter Lab ou Google Colab
2. Leia o enunciado da questão
3. Implemente a solução na célula indicada
4. Execute e teste seu código

## Observações

- Cada questão é independente
- Priorize clareza e organização do código
- Justifique as escolhas de estruturas de dados quando necessário
